Programa em Linguagem Python - SAV

Os prints do programa abaixo são referentes à entrega da matéria Computational Thinking Using Python para o primeiro Sprint. Foi requisitada a implementação de uma rotina contendo um Menu com as funcionalidades de maior relevancia para o projeto. Para cada funcionalidade definida, foi pedida uma implementação inical, contendo as regras do negocio planejada para as funcionalidades em questão. Tudo isso considerando os conceitos de entrada, processamento (simples) e saída de dados.

Nesse primeiro prototipo do programa foram implantadas as seguintes funções:

  • Entrada e Saída de dados
  • Processamento simples dos dados
  • Estruturas condicionais de repetição
  • Armazenamento de dados em variaveis locais
  • Resumo da operação realizada
  • Permissão para o usuário realizar uma nova operação
