Programa em Linguagem Python - SAV

O s prints do programa abaixo é referente a entrega da matéria Computational Thinking Using Python para o primeiro Sprint. É necessário implementar uma rotina contendo um Menu com as funcionalidades de maior relevancia para o projeto. Para cada funcionalidades definida é necessário que haja implementação inical contendo as regras do negocio planejada para as funcionalidades em questão, considerando os conceitos de entrada, processamento (simples) e saída de dados.

Nesse primeiro prototipo do programa foram implantadas as seguintes funções:

  • Entrada e Saída de dados
  • Processamento simples dos dados
  • Estruturas condicionais de repetição
  • Armazenamento de dados em variaveis locais
  • Resumo da operação realizada
  • Permissão para o usuário realizar uma nova operação
