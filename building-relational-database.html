Diagrama do Banco de Dados - SAV

O Diagrama abaixo é referente a primeira entrega da matéria Building Relational Database para o primeiro Sptint. É necessário um Diagrama que relacionasse as principais entidades e atributos para a solução da proposta, sendo composto por no mínimo 5 entidades com 4 atributos para cada uma. Com base na nossa necessidade inicial propomos 7 entidades, que muito provavelmente será modificado e / ou adaptado ao longo do desenvolvimento deste projeto.

As Entidades até o momento são:

  • Principal - Como a entidade principal
  • Cliente
  • Endereço
  • Documentos
  • Bicicleta
  • Modalidade
  • Seguro
