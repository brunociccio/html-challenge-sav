Diagrama de Classes do Projeto SAV

O Diagrama abaixo é referente a entrega da matéria Driven Design para o primeiro Sprint. É necessário desenvolver um Diagrama que fosse composto por pelo menos 8 a 10 classes inicialmente. Com base nisso estabelecemos 8 classes para dar um start em nosso Projeto, que muito provavelmente será adaptado ao longo do desenvolvimento do mesmo com novas classes.

As classes até o momento são:

  • Sistema - Como a classe principal
  • Cliente
  • Endereço
  • Documentos
  • Contato
  • Bicicleta
  • Modalidade
  • Seguro
